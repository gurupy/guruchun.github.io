## 종합 문제 1

* Armstrong Number
  - 각 자리수의 세제곱의 합이 자신과 같은 수
  - 100 ~ 999 사이에서 찾아 출력
* 제곱의 합과 합의 제곱
* 구구단의 합
* IP 주소 출력 (bin -> dec)
  - 2진수 포맷 문자열 32글자를 입력받아
  - 8글자씩 나눈 후
  - 10진수로 변환해 a.b.c.d 포맷으로 출력
* 시저의 암호
  - 문자열을 입력받고
  - 각 문자를 소문자로 변환하고
  - 각 문자의 코드값에 3을 더해 출력하기
