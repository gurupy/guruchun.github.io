```python

```
* 클래스와 객체의 맴버 변수
```python
class MyClass:
    my_name = "class"
    def say(self, _message):
        print(_message)
        print(MyClass.my_name)
        
    @classmethod
    def say_name(cls):
        print("my name is {}".format(cls.my_name))       
         
obj = MyClass()
obj.say("hi! ")
MyClass.say_name()

```

* 생성자와 소멸자
```python
class MyClass:
    def __init__(self, _family):
        print ("class created --->")
        
    def __del__(self):
        print ("class destroyed --->")
```
